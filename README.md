 Customer Churn Prediction

 Overview
Machine Learning project that predicts whether a customer is likely to leave a telecom company.

 Business Objective
Customer retention is significantly cheaper than acquiring new customers. This model helps identify at-risk customers and improve retention strategies.

Features
- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- Churn Prediction
- Streamlit Dashboard

Tech Stack
- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit

Results
- Random Forest Classifier
- Accuracy: 85%+

Future Improvements
- XGBoost
- Hyperparameter Tuning
- Customer Segmentation
- Real-Time Predictions
